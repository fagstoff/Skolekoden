Slik kan du bidra¶

Ønsker du å bidra med innholdsproduksjon kan det enkelt gjøres ved å:

  1. Lag en fork på Github

  2. Gjør endringer/lag innhold

  3. Legg inn en pull request på Github

  4. Følg retningslinjene som er beskrevet under

Retningslinjer for utforming av fagstoff-sider¶

Alle fagstoffsider skal oppfylle kravene som er gitt nedenfor.

Generelle krav¶

  • Alle oppgaver skal implementeres som Jupyter notebooks eller markdownfiler

  • Alle fagstoffsideer skal merkes med en Creative Commons-lisens

    • Foretrukket lisens er CC BY-SA

Krav til innhold¶

  • Fagteksten skal starte med en kort ingress som gir en introduksjon til fagstoffet som skal behandles

  • Python-koden bør ligge så nært opp til standarden PEP-8 som mulig

  • Matematikk skal være typesatt med LaTeX

  • Fagteksten skal avsluttes med oppgaver
